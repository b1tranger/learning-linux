In Linux, tput is a command used to control and customize the terminal screen. Instead of hardcoding weird symbols (ANSI escape codes), tput gives you a simple way to move the cursor, change colors, and clear the screen [2, 3].
Think of it as the "Remote Control" for your terminal window [5].
## Common tput Actions:

| Command | Action |
|---|---|
| tput cup <y> <x> | Moves the cursor to a specific row (y) and column (x) [2, 5]. |
| tput clear | Clears the screen completely (same as the clear command) [1, 2]. |
| tput civis | Hides the cursor (makes it invisible—great for games) [4]. |
| tput cnorm | Shows the cursor again (restores it to normal) [4]. |
| tput setaf <0-7> | Changes text color (e.g., setaf 2 is green) [1, 3]. |
| tput sgr0 | Resets all styles (removes colors and bolding) [1, 3]. |
| tput bold | Makes the text bold [3]. |

## Advanced tput Concepts & Commands:

Beyond the basics used in `project.sh`, `tput` offers even more control for complex terminal interfaces:

| Command | Action |
|---|---|
| `tput setab <0-7>` | Changes the **background** color of the text. |
| `tput sc` / `tput rc` | **Save Cursor** (`sc`) and **Restore Cursor** (`rc`). Great for jumping to another area, printing something, and returning the cursor to its previous location without recalculating coordinates. |
| `tput el` | **Erase Line**: Clears everything from the cursor's current position to the end of the line. Useful for clearing old text without redrawing the whole screen. |
| `tput ed` | **Erase Down**: Clears everything from the cursor down to the bottom of the terminal screen. |
| `tput smul` / `tput rmul` | Start underline (`smul`) and remove underline (`rmul`). |
| `tput blink` | Makes the text blink (often used for urgent warnings or alerts). |
| `tput rev` | Reverses the foreground and background colors (creates a highlighted block effect). |
| `tput lines` & `tput cols` | Outputs the terminal's current height (lines) and width (columns). Extremely useful for making games dynamically resize to fit the player's window. |
## Why use it in your game?
In your project, tput is essential because it allows you to:

   1. Animate: You move the cursor back to the character's position to redraw them without clearing the whole screen [2, 5].
   2. Professionalism: Hiding the blinking cursor (civis) makes your game feel like a real application rather than just a running script [4].
   3. Visuals: You can color-code your "Exit Door" (Red) and "Spawn Point" (Green) using setaf [3].
