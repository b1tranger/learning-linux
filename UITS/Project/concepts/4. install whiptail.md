To prepare your environment for developing the "Tavern of Life," follow this guide to verify and install **whiptail**, which is essential for creating the game's retro-style dialogue boxes and menus.

### 1. Check for Pre-installation
On many Linux distributions, especially Debian-based ones like Ubuntu, **whiptail** is often **pre-installed** because it serves as a dependency for various system configuration tools. 

To see if it is already available on your system, run the following command in your terminal:
```bash
whiptail --version
```
If this command returns a version number, you are ready to go and no further setup is required,.

### 2. Installation by Distribution
If the command above returns an error like "command not found," you will need to install it using your system's package manager.

*   **For Ubuntu / Debian / GitHub Codespaces:**
    ```bash
    sudo apt update
    sudo apt install whiptail
    ```
*   **For Fedora / CentOS / RHEL:**
    Whiptail is often included in the `newt` package on these systems:
    ```bash
    sudo dnf install newt
    ```
*   **For Arch Linux:**
    ```bash
    sudo pacman -S libnewt
    ```

### 3. Final Verification
After installation, run `whiptail --version` again to confirm the process was successful. Once available, you can call it directly from your Bash scripts to generate message boxes (`--msgbox`) or yes/no prompts (`--yesno`) as seen in your game's NPC logic.
