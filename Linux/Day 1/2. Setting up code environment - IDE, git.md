- installing Antigravity (instead of VS code) first:
    > https://antigravity.google/download/linux
    > 
    Add the repository to sources.list.d
    ```
    sudo mkdir -p /etc/apt/keyrings
    curl -fsSL https://us-central1-apt.pkg.dev/doc/repo-signing-key.gpg | \
    sudo gpg --dearmor --yes -o /etc/apt/keyrings/antigravity-repo-key.gpg
    echo "deb [signed-by=/etc/apt/keyrings/antigravity-repo-key.gpg] https://us-central1-apt.pkg.dev/projects/antigravity-auto-updater-dev/ antigravity-debian main" | \
    sudo tee /etc/apt/sources.list.d/antigravity.list > /dev/null
    ```

    What is Curl Command in Linux?

    Curl (Client URL) is a command-line tool used for transferring data with various protocols, including HTTP, HTTPS, FTP, and many more. <br>
    Ubuntu, being one of the most popular Linux distributions, often has curl available in its software repositories.  <br>
    This allows users to easily install and update curl using Ubuntu’s package management tools like apt. <br>

    Install curl.
    ```
    sudo apt install curl
    ```
    Verify Installation
    ```
    curl --version
    ```
